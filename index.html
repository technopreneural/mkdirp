Mkdirp

An implementation of the command `mkdir -p` in node js

Download .zip Download .tar.gz View on GitHub

Overview

This project aims to implement the functionality of the Linux/Unix CLI command mkdir -p in Node JS as a learning exercise.

Motivation

Just like most other developers, we found our web project workflow required a lot of automated file manipulation. Copying, moving, renaming, and deleting files and folders. This node module allows us to programmatically create a folder as well as the path to it if the path doesn't exist.

History

In the implementation of a general purpose automated build workflow for web projects (i.e. that can be used as a template for other projects), we initially opted to use Grunt and eventually settled with Gulp. Both are based on Javascript and both relied heavily on other packages to support functionality required by common web project workflows. This project attempts to deliver a module that provides one such functionality.
